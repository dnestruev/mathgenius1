MathGenius
MathGenius — это высокопроизводительный математический движок с гибридной архитектурой. Проект объединяет собственные алгоритмические решения для быстрых вычислений и мощь нейросети NVIDIA Nemotron-3 Nano Omni (30B) для пошагового разбора сложных задач.

Основные возможности
Гибридный вычислитель: Автоматическое распределение нагрузки между локальным математическим ядром и LLM.

Пошаговый интеллект: Нейросетевой разбор задач с представлением логических цепочек решения.

Архитектурная чистота: Оптимизированный код без лишних зависимостей.

Интерфейс: Минималистичный дизайн, вдохновленный современной эстетикой гаджетов, с акцентом на удобство и визуальную чистоту.

Технологический стек
Core Engine: math.js

AI Inference: NVIDIA Nemotron-3 Nano Omni (30B)

Frontend: Оптимизированный JavaScript и CSS
