MathGenius
MathGenius — это высокопроизводительный математический движок с гибридной архитектурой. Проект объединяет собственные алгоритмические решения для быстрых вычислений и интеллектуальную мощь модели NVIDIA Nemotron-3 Nano Omni (30B) для глубокого пошагового разбора задач.

Основные возможности
Гибридный вычислитель: Интеллектуальное распределение нагрузки между локальным математическим ядром и нейросетевым слоем.

Пошаговый интеллект: Структурированный разбор задач с отображением логических этапов решения.

Архитектурная чистота: Оптимизированный код без избыточных зависимостей.

Эстетичный интерфейс: Минималистичный дизайн, вдохновленный современной индустриальной эстетикой (Nothing/CMF), с акцентом на визуальную чистоту и отзывчивость.

Технологический стек
Core Engine: math.js

AI Inference: NVIDIA Nemotron-3 Nano Omni (30B) via OpenRouter API

Frontend: Оптимизированный JavaScript и CSS

Настройка API
Для работы нейросетевых функций требуется личный API-ключ OpenRouter. Проект настроен на использование бесплатных моделей, поэтому пополнение баланса не требуется.

Регистрация: Создайте аккаунт на OpenRouter.

Генерация ключа:

Перейдите в раздел Keys в настройках профиля.

Нажмите Create Key.

Укажите любое имя для ключа (например, MathGenius) и нажмите Create.

Сохранение: Скопируйте полученную строку (начинается с sk-or-v1-...). Внимание: ключ отображается только один раз.

Активация: Откройте файл mathgenius_v1.html в браузере. Вставьте ключ в соответствующее поле настройки и сохраните его.

💡 Ключ будет работать бесконечно, так как программа использует только бесплатные модели с пометкой :free.
